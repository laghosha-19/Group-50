    რეპი არის მუსიკალური ჟანრი, რომელიც თავის სათავეს იღებს ქუჩის კულტურაში და ძირითადად გამოირჩევა ლირიკული ექსპრესიულობითა და სიტყვების სწრაფი რითმული წარმოთქმით. რეპის ტექსტები ხშირად გადმოსცემს ემოციებს, სოციალურ და კულტურულ თემებს, ისტორიებს და აზრებს, რომელთა საშუალებითაც არტისტები გამოხატავენ პირად გამოცდილებასა და გარემოსთან დამოკიდებულებას. რეპი ხშირად მოიცავს ხაზებს, რომლებიც აღიარებს ცხოვრებისეული სირთულეების გავლენას, საზოგადოების პრობლემებს, პოლიტიკური იდეალებს და პიროვნულ ბრძოლებს. პროექტში შეიძლება ასახულ იქნას როგორც მელოდიური, ისე უფრო მძიმე, მქუხარე რეპის სტილი, რომლის მიზანიც არა მხოლოდ მუსიკალური გამოცდილება, არამედ სოციალური და კულტურული ცვალებადობების შეხედულებაა.
